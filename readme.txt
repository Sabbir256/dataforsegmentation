This dataset contains 3 types of tumor images.
They are:
	- meningioma
	- glioma and 
	- pituitary tumor

The labels are 1 for meningioma, 2 for glioma, 3 for pituitary tumor.

tumorMask dataset contains only the tumor regions of data. These are binary images.

Total 3064 T1 weighted images of 233 different patients are there in the dataset folder in (.png) format.
